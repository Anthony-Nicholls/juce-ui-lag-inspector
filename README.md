# JUCE UI Lag Inspector

## What is it?
A free and lightweight reference plugin that can be loaded into a host to inspect UI performance and timing for JUCE-based plugins.

If UI performance issues are discovered, this plugin provides an easy-to-share reference that anyone can load to reproduce and report the problem.

## Supported plugin formats
- AAX
- AU
- AUv3
- Standalone
- VST3

## Supported operating systems
- **iOS**: iOS 12+
- **Linux**: Mainstream distributions
- **macOS**: macOS 10.11+
- **Windows**: Windows 10+
